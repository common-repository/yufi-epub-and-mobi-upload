Allow ePUB and MOBI file uploads.

== Description ==

Simple plugin enabling both ePub and MOBI file format uploads.

It simply adds these mine types to the existing allowed mine types.

Has been tested on both single and multisite WordPress installs.

== Installation ==

Standard plugin installation.  Upload the files to your server using the built in plugin installer or directly to the servers plugin folder.

Activate the plugin.
